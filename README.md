# AstroBastion

Tower Defense с action-элементами в космическом сеттинге. Игрок обороняет планету, размещая башни на кубических узлах в трёхмерном пространстве, и параллельно управляет боевым кораблём, участвуя в сражении.

Враги перемещаются по объёмным 3D-траекториям (сплайнам). Игрок может расширять сеть обороны, формируя огневые рубежи.

**Прототип сделан за 1.5 месяца в одиночку**

## Отдельные технические решения

### Object Pooling
Возврат объектов за O(1) через swap-логику вместо поиска по массиву и динамическое расширения пула при необходимости.

### Визуализация наносимого урона
Наносимый врагам урон визуализируется не как элемент интерфейса, а как набор трехмерных объектов-цифр. Для оптимизации используется gpu instancing. 

### State Machine для контекстного UI
Интерфейс управления системой строительства и башнями реализован через состояния. Каждое состояние хранится как ScriptableObject и определяет набор доступных кнопок и действий. Переключение контекста происходит при выборе кнопки в контрольной панели.

### Система формаций
Гибкая система планирования волн позволяет создавать не только определенное количество врагов подряд, но и формации — группы врагов с заданным расположением в пространстве и индивидуальными параметрами каждого юнита, что упрощает создание и настройку сложных волн.

## Структура

```
Scripts/
├── Buildable/        — башни
├── Builder/          — строительство и размещение объектов
├── Enemy/            — враги, визуализация урона
├── Player/           — управление кораблём
├── Portal/           — траектории врагов
├── Projectiles/      — снаряды
├── UI/               — меню строительства, стейт-машина, HUD
├── Warlord/          — управление волнами, формации
├── Localizer/        — система локализации
└── Toolkits/         — утилиты
```
